# Form Builder

Form Builder is php 5 class to create html5 form, supports :

- input
  - text, password, email, tel, search
  - radio
  - checkbox
  - file
  - number, range
  - color
  - date, datetime, datetime-local, time, month, week
  - url
  - button, submit, reset

- select
  - single
  - multiple
  - combo

- fieldset
- htmlarea
- paragraph
- button
