# Bootstrap Form Builder

Form Builder is php 5 class to create Twitter Bootstrap html5 form, supports :

- input
  - text, password, email, tel, search
  - radio
  - checkbox
  - file
  - number, range
  - color
  - date, datetime, datetime-local, time, month, week
  - url
  - hidden
  - button, submit, reset

- datalist
 
- select
  - single
  - multiple
  - combo

- fieldset
- htmlarea
- paragraph
- section
- button
