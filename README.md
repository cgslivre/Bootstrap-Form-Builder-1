# Form Builder

Form Builder is php 5 class to create html5 form, supports :

- input
  - text
  - radio
  - checkbox
  - password
  - file
  - number
  - range
  - email
  - search
  - tel
  - color
  - month
  - week
  - date
  - datetime
  - datetime-local
  - url
  - button
  - submit
  - reset
  
- button

- select
  - single
  - multiple
  - combo

- paragraphe

- fieldset

- htmlarea
